# Contributing

This public repository is source-available for visibility, use, review, and
issue reporting. It is not an open contribution project.

Pull requests and patches are not accepted unless they were requested or
approved in writing by morcoan before submission. If a contribution is accepted,
the contributor must agree to the license and any additional contribution terms
requested for that change.

For bug reports, include:

- the command that was run;
- the relevant platform and Rust version;
- whether optional Cargo features were enabled; and
- a minimal input or synthetic reproduction when possible.
